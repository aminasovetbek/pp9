# Mickey's Clock 

A Pygame-based analog clock using Mickey Mouse hand graphics.

## Features
- Displays current system time (minutes & seconds)
- White glove = **Minutes hand**
- Yellow glove = **Seconds hand**
- Mickey Mouse face on the clock dial
- Digital MM:SS display at the bottom
- Real-time synchronization with system clock

## How to Run

```bash
pip install pygame
python main.py
```

## Controls
- **ESC** or close window to quit
