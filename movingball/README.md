# Moving Ball Game 

A simple Pygame game with a moving red ball.

## Controls
| Key | Action |
|-----|--------|
| ↑ | Move up |
| ↓ | Move down |
| ← | Move left |
| → | Move right |
| ESC | Quit |

## How to Run
```bash
py -3.12 main.py
```

## File Structure
```
moving_ball/
├── main.py   # Game loop and drawing
├── ball.py   # Ball class with movement and boundary checking
└── README.md
```
